# ML Evaluation — Interactive Textbook Collection

Interactive HTML visualisations and textbooks for machine learning evaluation, covering three core exam topics: **metrics**, **sampling & error estimation**, and **significance testing**.

These projects are based on the following sources:

- Demšar, J. (2006). *Statistical Comparisons of Classifiers over Multiple Data Sets*. Journal of Machine Learning Research, 7, 1–30.
- Flach, P. A. (2012). *ROC Analysis* (ICML 2012 Tutorial).
- Additional course materials and lecture notes from the machine learning evaluation curriculum.
