What “local” means

A curved isometric can have one slope here and another slope somewhere else. “Local slope” means: zoom in very close to one selected point and replace the curve by its tangent line.

For a straight isometric, the local slope is the same everywhere. For a curved isometric, it changes from point to point.

4. Gradient intuition

Along an equal-value curve, the metric cannot change: \[ dM=M_x\,dx+M_y\,dy=0. \] Therefore: \[ \frac{dy}{dx}=-\frac{M_x}{M_y}. \] Here \(x=FPR\), \(y=TPR\). \(M_x\) asks how the metric changes if only FPR moves; \(M_y\) asks how it changes if only TPR moves.
